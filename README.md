# SCsProjectMaster

Für die Verwendung der App ist eine Internetverbindung mit HSMW-VPN notwendig.
Für die Verwendung der App wird Windows 10 empfolen. Nutzen sie hierfür
SCsProjectMaster\bin\Release\net7.0-windows10.0.19041.0\win10-x64\AppPackages\SCsProjectMaster_1.0.2.0_Test\SCsProjectMaster_1.0.2.0_x64.msix
zum installieren der App.

Achtung:
Nicht die .exe verwenden da die Anwendung nur self contained als .msix funktioniert.
